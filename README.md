# React Router 직접 만들어보기

## 목표

```tsx
<Router>
  <Route path="/" component={<Root />} />
  <Route path="/about" component={<About />} />
</Router>
```

1. 위 코드가 동작하도록 `<Router>`와 `<Route>` 컴포넌트를 만든다.
2. 하나의 페이지에서 다른 페이지로 이동할 수 있다.
3. 브라우저 인터페이스(뒤로가기/앞으로가기 버튼)를 이용해 페이지를 이동할 수 있다.
